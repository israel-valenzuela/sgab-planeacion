# Estimación del Proyecto

## Suposiciones
- El sistema será desarrollado por un equipo pequeño.
- No se integrará con sistemas externos.
- Se trata de un sistema web básico.

## Descomposición funcional

| Módulo                | Descripción                   | Horas estimadas |
|-----------------------|------------- -----------------|-----------------|
| Gestión de usuarios   | Registro y control de accesos | 30 |
| Gestión de alumnos    | Registro y consulta           | 25 |
| Gestión de materias   | Administración de materias    | 20 |
| Calificaciones        | Registro y consulta           | 35 |
| Interfaz general      | Diseño del sistema            | 30 |

## Estimacion de esfuerz: 140 horas/ 3 persona.

## Nivel de incertidumbre
La estimación tiene un nivel medio de incertidumbre, ya que pueden surgir cambios en requisitos o ajustes en el diseño durante el desarrollo.
