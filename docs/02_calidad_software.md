# Calidad del Software

Para el sistema SGAB se consideran los siguientes atributos de calidad como los más importantes:

## Seguridad
La seguridad es fundamental porque el sistema manejará información sensible como datos personales y calificaciones de los alumnos.  
Si no se cumple este atributo, podría existir acceso no autorizado, pérdida de información o manipulación de datos académicos.

## Usabilidad
El sistema debe ser fácil de usar para administradores, docentes y alumnos.  
Si no se garantiza una buena usabilidad, los usuarios podrían cometer errores, tardar más en realizar tareas y rechazar el uso del sistema.

## Confiabilidad
El sistema debe funcionar correctamente y mantener la información consistente.  
Si no se cumple este atributo, podrían presentarse fallas, pérdida de datos o información incorrecta, afectando procesos académicos.
