# Descripción del Proyecto

## Objetivo del sistema: 
El Sistema de Gestión Académica Básico (SGAB) tiene como objetivo administrar y centralizar la información académica de una institución educativa, permitiendo el control eficiente de alumnos, materias, calificaciones y usuarios.

## Usuarios principales
El sistema contempla los siguientes tipos de usuarios:
- Administrador*: gestiona usuarios, materias y configuración general del sistema.
- Docente: registra y consulta calificaciones de los alumnos asignados.

## Problema que resuelve:
En muchas instituciones educativas, la gestión académica se realiza mediante hojas de cálculo, documentos físicos o sistemas poco integrados, lo que provoca duplicidad de información, errores en el registro de calificaciones, dificultad para consultar datos históricos y pérdida de tiempo en tareas administrativas repetitivas.
El SGAB resuelve estos problemas al proporcionar una plataforma centralizada, accesible y estructurada para la administración académica básica.

## Suposiciones iniciales
Para la planeación del sistema se asumen las siguientes condiciones:
- El sistema será utilizado por una sola institución educativa.
- El número de usuarios será moderado (instituciones pequeñas o medianas).
- El acceso al sistema se realizará mediante un navegador web.
- No se contempla integración con sistemas externos en la versión inicial.
