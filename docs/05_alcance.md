# Definición del Alcance

## Funcionalidades incluidas
- Gestionar alumnos
- Materias
- Alumnos
- Calificaciones
- Reportes basicos
- sitema academico sin interaciones ecternas

## Funcionalidades excluidas
- Pagos en línea
- Integraciones gubernamentales
- Aplicación móvil

## Suposiciones
- El sistema será usado por una sola institución.
- Los usuarios tendrán acceso a internet.
- El sistema será web.

## Ejemplo de cambio y su impacto
Si se solicita agregar pagos en línea, se requeriría modificar el diseño del sistema, aumentar el tiempo de desarrollo y considerar aspectos de seguridad adicionales.
