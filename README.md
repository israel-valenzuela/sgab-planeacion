# Sistema de Gestión Académica Básico (SGAB)

## Integrantes del equipo: Israel Valenzuela Razo 


## Descripción:
El Sistema de Gestión Académica Básico (SGAB) es un sistema web diseñado para administrar
de manera centralizada la información académica de una institución educativa.  
El sistema permite gestionar alumnos, materias, calificaciones y usuarios, facilitando el
 registro, consulta y mantenimiento de la información académica.

SGAB está orientado a instituciones educativas pequeñas o medianas que requieren una solución 
sencilla, clara y confiable para el control de sus procesos académicos básicos.

## Objetivo:
El objetivo principal del SGAB es mejorar la organización y el acceso a la información académica,
 reduciendo errores manuales, optimizando tiempos administrativos y proporcionando una base
  estructurada para futuras ampliaciones del sistema.

## Alcance del repositorio
Este repositorio contiene la planeación inicial del proyecto SGAB, incluyendo el análisis 
previo al desarrollo del software.  
Se documentan aspectos clave como:

- Descripción del problema y del sistema
- Atributos de calidad del software
- Estimación inicial de esfuerzo
- Identificación y mitigación de riesgos
- Definición clara del alcance del sistema

El repositorio no incluye código fuente, ya que su finalidad es establecer una base sólida
de análisis y toma de decisiones antes de la implementación.
